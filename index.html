Popup32 - VBScript Popup Tool

About Popup32

Popup32 is a small VBScript project that allows you to create multiple message boxes with custom titles, messages, and types. You can enable an optional 10-second cooldown between popups. It's a safe, lightweight tool for testing or fun notifications.

Features

  • Create multiple message boxes with custom titles and messages
  • Choose from different popup types (Critical, Exclamation, Information, Question, Plain)
  • Optional cooldown between popups
